How it works

Zero effort. Automatic.

Every Claude Code session ends with a Stop hook that silently extracts, embeds, and stores facts in the background — so Claude remembers without you doing anything.

01

Session ends

Claude Code's Stop hook fires at the end of every session. It reads the last 4,000 characters of the transcript JSONL.

02

LLM extracts facts

llama3.2:3b runs locally via Ollama, extracting only facts worth remembering long-term — preferences, decisions, project details.

03

Semantic dedup

Each fact is embedded with nomic-embed-text. Cosine similarity at 0.92 threshold prevents near-duplicates from piling up.

04

Stored locally

New facts land in ~/.abhimem/memory.db — a local SQLite database. No cloud, no API keys, no accounts.

05

Available next session

A MEMORY.md index loads automatically. Mid-session, Claude calls recall to semantic-search across all stored memories.

Features

Smart. Silent. Local.

Everything runs on your machine using open-source models. abhimem never phones home.

⚡

Non-blocking extraction

The Stop hook exits in milliseconds. Extraction runs in a detached subprocess so it never slows down your Claude Code responses.

◎

Semantic deduplication

Vector similarity at 0.92 threshold. If a fact is already known, it's skipped. Your memory stays clean, not noisy.

MCP Tools

Six tools. All in Claude.

Once installed, these tools are available in every Claude Code session via the FastMCP server running on localhost:8765.

Tool Description
mcp__abhimem__remember Store a memory manually. Params: text, category (preference / task / project / person / general), source.
mcp__abhimem__recall Semantic search over all memories. Returns top matches with similarity scores. Params: query, limit.
mcp__abhimem__list_memories List recent memories, optionally filtered by category. Returns paginated results.
mcp__abhimem__forget Delete a specific memory by ID. Use after a daily review to remove junk or outdated facts.
mcp__abhimem__sync_to_obsidian Export all memories to Claude Memory.md in your Obsidian vault, grouped by category.
mcp__abhimem__import_from_obsidian Import .md files from your Obsidian Memory Inbox/ folder, embedded and stored automatically.
Install

One command.

Requires macOS, Python 3.9+, and Ollama. The installer handles everything else — models, LaunchAgent, MCP registration, Stop hook.

SHELL install.sh
# clone and install
git clone https://github.com/AbhiPoluri/abhimem
cd abhimem && bash install.sh

# restart Claude Code to activate

# data lives here — always local
~/.abhimem/memory.db

# MCP server auto-starts on login at
http://127.0.0.1:8765/mcp
